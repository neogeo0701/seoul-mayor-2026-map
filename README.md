# 2026 제9회 지방선거 — 서울특별시장 자치구별 득표 단계구분도

🔗 **배포 URL: https://neogeo0701.github.io/seoul-mayor-2026-map/**

TMAP 지도 위에 서울 25개 자치구별 **오세훈(국민의힘) vs 정원오(더불어민주당)** 득표
격차를 다이버징 단계구분도(choropleth)로 그린 단일 페이지 웹지도입니다.
대학 지도학 과제용.

**주요 기능:** 다크 헤더바·요약 스탯카드 · 자치구 이름 라벨 · 구 클릭 시 확대 + 디테일
패널(득표 막대·격차·랭킹) · 좌하단 격차 랭킹 바차트(지도 연동) · 분류법 토글(자연분류/등간격/분위수)
· 격차 히스토그램 · 인트로 스플래시.

---

## 실행 방법

로컬 HTTP 서버가 필요합니다(브라우저 보안상 `file://`로 열면 `data/` 로드가 막힙니다).

```bash
cd seoul-election-map
python -m http.server 5500
```

브라우저에서 **http://localhost:5500** 접속.

---

## TMAP AppKey 넣기 (한 줄)

`index.html` 최상단의 이 한 줄만 본인 키로 바꾸면 됩니다.

```js
const TMAP_APP_KEY = ""; // <-- 여기에 AppKey 붙여넣기
```

- 키 발급: **SK open api** (https://openapi.sk.com) → 회원가입 → 앱 등록 → TMAP API 신청 → AppKey 복사
- **반드시 "Any IP allowed"(모든 IP 허용)로 설정**하세요. (localhost 테스트 시 IP 제한이 걸리면 지도가 안 뜹니다.)
- 키가 **비어 있어도** 페이지는 동작합니다 → TMAP 베이스맵 대신 **SVG 미리보기**로 25개
  폴리곤·색칠·범례·툴팁이 그대로 보입니다. 키를 넣으면 TMAP 실제 지도 위에 렌더링됩니다.

## 기능

- **베이스맵**: TMAP JS Map SDK (`tmap/jsv2`) 위에 25개 자치구 폴리곤 오버레이
  (키 없을 때는 Web Mercator 투영 SVG 미리보기로 자동 대체)
- **색칠**: 다이버징 — 오세훈 우세=주황 계열, 정원오 우세=파랑 계열, 진할수록 격차 큼.
  순수 원색 빨강·파랑은 *chromostereopsis*(빨강 돌출 착시)를 유발하므로 **톤다운한 6색** 사용.
- **분류법 토글**: 자연분류(Jenks, 기본) / 등간격 / 분위수 — 좌상단 버튼.
- **범례**: 6단계 색 + 격차(%p) 구간 + 25개 구 격차 **히스토그램**(0=동률 기준선 포함).
- **툴팁**: 구 hover 시 구명·오세훈/정원오 득표수·득표율·격차 표시.

## 데이터

- **경계**: `data/seoul_municipalities.geojson`
  (southkorea/seoul-maps, KOSTAT 2013, WGS84 좌표계 — 재투영 불필요, 25개 구)
- **득표**: `data/votes.json` — **실데이터**.
  중앙선거관리위원회 개표결과를 정리한 집계표(나무위키 「제9회 전국동시지방선거/지역별 결과/수도권/서울특별시」, 2026-06-06 취득) 기준.

### 데이터 검증(검산) 결과
- 25개 구, **오세훈 우세 10구 : 정원오 우세 15구** — 공식과 일치 ✅
- 강남 **오세훈 +99,598표**, 서초 **+73,028표** — 공식 검산값과 **정확히 일치** ✅
- 구별 합계: 오세훈 2,572,014 / 정원오 2,508,705 — 공식 최종합계(2,575,819 / 2,515,560)의 **99.9%** 수준
- ⚠️ **알려진 소폭 차이**: 송파 격차가 데이터상 +47,512표(공식 검산값 +40,713표), 전체격차 +63,309표(공식 +60,259표)로
  약간 다릅니다. 취득 스냅샷 시점 차이로 추정됩니다. 공간 패턴(우세 후보·강도)은 정확합니다.

> 더 정밀한 수치가 필요하면 **info.nec.go.kr**(중앙선관위 선거통계시스템) →
> 제9회 전국동시지방선거 → 시·도지사 → 서울특별시 → 자치구별 표를 복사해
> `data/votes.json`의 각 구 `oh.votes` / `jung.votes` / `oh.pct` / `jung.pct`를 교체하세요.
> 키(구 이름)는 GeoJSON `properties.name`과 1:1로 맞아 있습니다.
